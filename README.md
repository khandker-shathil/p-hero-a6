# Digital Tools Buying Website

---

## Description
A simple React website where digital products are showcased. The user can add products to their cart and checkout.

---

## Tech Used
- React.js  
- Tailwind CSS  
- DaisyUI  
- JavaScript (ES6+)  
- React-Toastify  
- JSON  

---

## Features
1. Adding products to the cart.  
2. Removing products from the cart.  
3. 'Proceed to Checkout' button clears the cart.
